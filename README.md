# 🪦 Domain Graveyard

**Domain Graveyard** is a small web app that checks whether a domain name is already registered and presents the result as a humorous “life or death” report for your startup idea.

If the domain is taken, it gets a proper obituary. If it’s available, it gets a rare second chance at life.

---

## 💡 Idea

Most startup ideas begin with a domain check.

This project turns that moment into a joke:

- Registered domain → 💀 “Already dead”
- Available domain → ✨ “Still alive”
